## Clone URL

1. Clone this repository using the following URL:

```bash
git clone https://github.com/tushii05/bulk_email_excelsheet.git
```
2. Navigate to the project directory:

```bash
cd bulk_email_excelsheet
```

3. Install dependencies using npm:

```bash
npm install
```
